# random-chat
Socket.io, node chat app 

## Clone: 
```sh
 git clone https://github.com/disko998/random-chat.git
 ```

## How to run it?

install [nodemon](https://www.npmjs.com/package/nodemon) first

```sh
$ cd random-chat
$ npm install
$ npm run start
$ open index.html
```
